# Strata

**Strata** は、途中の思考や迷いを、そのまま残すための  
ローカルファーストなツールです。

整理しなくていい。  
結論を出さなくていい。  
あとから、文脈として振り返れれば十分です。

---

## Strata とは

Strata は、メモアプリでも、日記アプリでも、AIコーチでもありません。

- 考えがまとまっていない
- 行動しながら迷っている
- 公開したくないけれど、消したくもない

そんな「途中の状態」を、気負わずに置いておくための場所です。

---

## できること（初期MVP）

- 付箋のような小さな入力画面に、思いついたことを書く
- 入力した内容はすべてローカルに保存される
- 日付順のタイムラインで、過去の思考を振り返る
- 何も発信しなくてよい

将来的に、AIによる整理や振り返りの補助を追加する予定ですが、  
それは常に「必要なときだけ」「明示的に」使える形を想定しています。

---

## 設計思想

- **ローカルファースト**  
  データは端末内に保存され、外部に送信されません。

- **低い心理的負荷**  
  分類・タグ付け・目的設定は必須ではありません。

- **未完成を前提にする**  
  思考は完成させるものではなく、積み重なるものとして扱います。

- **答えより文脈**  
  正解を出すことよりも、「なぜそう考えていたか」を残します。

---

## 現在のステータス

Strata は個人開発の初期段階にあるプロジェクトです。

まずは、

- デスクトップアプリ（Tauri）
- 入力 → 保存 → 一覧表示

という最小の体験を丁寧に作ることを優先しています。

---

## 名前について

**Strata** は「地層」を意味します。

思考や迷いは、  
その場では意味を持たなくても、  
時間とともに層となり、文脈を形作ります。

Strata は、その層を静かに残すための場所です。
